# Custom Random Generator

## Описание
Небольшая программа на Python для генерации случайных чисел с использованием собственного класса `Random`.
Пользователь вводит диапазон после чего программа сгенерирует случайное число в указанном диапозоне.

## Технологии
- Python3
- Классы и методы

## Инструкция по запуску
1. Установите Python3.
2. Скачайте файл `custom_random.py`.
3. Откройте терминал в папке проекта.
4. Выполните команду:

```bash
python custom_random.py
```

5. Введите диапозон чисел для генерации
6. Получите случайное число в диапозоне
